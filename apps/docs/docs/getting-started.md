# Getting Started

Clock UI is a component library providing analog clock components for DOM, React, and Vue.

## Installation

Install the packages you need using your preferred package manager:

```bash
# npm
npm install @clock-ui/dom @clock-ui/react @clock-ui/vue

# yarn
yarn add @clock-ui/dom @clock-ui/react @clock-ui/vue

# pnpm
pnpm add @clock-ui/dom @clock-ui/react @clock-ui/vue

# bun
bun add @clock-ui/dom @clock-ui/react @clock-ui/vue
```

## Quick Start

Choose the package that matches your framework:

- [DOM Package](/docs/dom) - Vanilla JavaScript
- [React Package](/docs/react) - React components
- [Vue Package](/docs/vue) - Vue components

Each package includes both static and live clock components with extensive customization options.
